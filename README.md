# 行星秤重動畫 — 克卜勒二體問題模擬

**🌐 直接瀏覽 / Live Demo:** <https://xun73.github.io/planet-mass-sim/>

## 中文說明

這是一個克卜勒運動物理模擬網頁，展示**雙星系統繞質心互繞**的運動狀態。  
使用者可以調整兩天體的質量、距離、週期，即時觀察軌道運動與計算公式。

### 功能
- 調整 M（中心星）、m（伴星）質量
- 調整兩星距離 a 與公轉週期 T
- 觀測視角切換（正上方／傾角／側面）+ 滑鼠拖曳旋轉
- 縮放功能（滾輪）
- 即時顯示萬有引力、向心力計算結果
- M + m 公式推導展示
- 中英文雙語介面（自動依瀏覽器語言切換）

### 操作
| 控制 | 說明 |
|------|------|
| 滑桿 / 數字輸入 | 調整參數 |
| 快捷按鈕 | 一鍵設定行星質量 |
| 右鍵拖曳畫布 | 旋轉觀測視角 |
| 滾輪 | 縮放畫面 |
| 視角按鈕 | 快速切換俯視／側面 |

## English

An interactive physics simulation of **Kepler's two-body problem**, showing two celestial bodies orbiting their common barycenter. Adjust masses, distance, and period to see real-time orbital motion and calculations.

### Features
- Adjust M (central star) and m (companion) masses
- Adjust distance a and orbital period T
- View angle control (top / tilt / side) + drag to rotate
- Zoom with scroll wheel
- Real-time gravitational & centripetal force computation
- M + m formula derivation display
- Bilingual UI (Chinese / English, auto-detects browser language)

### Controls
| Control | Description |
|---------|-------------|
| Slider / Number input | Adjust parameters |
| Quick buttons | One-click planet mass presets |
| Right-drag on canvas | Rotate view angle |
| Scroll wheel | Zoom in/out |
| View buttons | Quick top/side view |
